Прототип (он же Клон). Порождающий паттерн.
https://metanit.com/sharp/patterns/2.4.php

Шаблон клонирования основываясь на каком-то готовом объекте. Клонируем этот объект с помощью реализованного у него метода. 
Есть несколько сущностей наследников, их надо уметь клонировать:

1) В их абстрактном классе/интерфейсе создаём метод: abstract Clone(), чем заставляем наследников его реализовать.
/
Ну или создаём отдельный интерфейс для общего рода сущностей, из которого придётся как то понимать с объектом какого-типа ты работаешь

# Надо реализовывать для объектов, для которых Фабрика не подходит по каким то разумным причинам.

Например, используется для фигур:
Прямоугольники, треугольники всякие - их надо уметь клонировать.
