.NET Host builder

Используется не только для создания сервисов/веб-сервисов, но и так же, обыкновенных приложений.
Не факт, но насколько я знаю, первоначально он был придуман для asp.net, как удобная прослойка для взятия запросов.

Универсальный хост, создаваётся и настраивается с помощью билдера (расширение IHost.CreateDefaultBuilder()).

Под собой настроенный хост отвечает за:
1. Внедрение зависимостей
2. Логгирование
3. Созданием и упарвлением сервисами, управление их жизнью
4. Позволяет читать и удобно пользоваться конфигурацией

Пример:

await Host.CreateDefaultBuilder(args)
	.ConfigureServices(() => 
	{
		services.AddHostedService<SomeMyService>();
	})
	.Build()
	.StartAsync();
